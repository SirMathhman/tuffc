# Tuff

A TypeScript project powered by Bun.

## Setup

```bash
# Install dependencies
bun install
```

## Development

```bash
# Run the development server
bun run dev
```

## Build

```bash
# Build the project
bun run build
```

## Production

```bash
# Start the production build
bun run start
```

## Testing

```bash
# Run tests
bun test
```

## Project Structure

```
src/
  index.ts          # Main entry point
dist/              # Build output (generated)
tsconfig.json      # TypeScript configuration
package.json       # Project configuration
```
