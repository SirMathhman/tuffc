# Tuff Project

A basic TypeScript project.

## Getting Started

### Prerequisites

- Bun (latest version)

### Installation

Install dependencies:

```bash
bun install
```

### Development

Build the project in watch mode:

```bash
bun run dev
```

### Build

Compile TypeScript to JavaScript:

```bash
bun run build
```

### Run

Execute the compiled code:

```bash
bun start
```
