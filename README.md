# Tuff Project

A basic TypeScript project.

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

Install dependencies:

```bash
npm install
```

### Development

Build the project in watch mode:

```bash
npm run dev
```

### Build

Compile TypeScript to JavaScript:

```bash
npm run build
```

### Run

Execute the compiled code:

```bash
npm start
```
